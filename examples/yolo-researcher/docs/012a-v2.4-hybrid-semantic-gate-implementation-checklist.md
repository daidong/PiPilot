## 9. Operator Notes

### 9.1 Enable in desktop runtime

Desktop now enables semantic gate by default (`enforce_touch_only`) and attempts live LLM arbitration automatically.
Use environment variables to override:

```bash
export YOLO_SEMANTIC_GATE_MODE=shadow   # or off / enforce_touch_only
export YOLO_SEMANTIC_GATE_CONFIDENCE=0.85
export YOLO_SEMANTIC_GATE_MAX_INPUT_CHARS=18000
export YOLO_SEMANTIC_GATE_MODEL=gpt-5.2
```

Supported modes:

- `off`
- `shadow`
- `enforce_touch_only`
- `enforce_success` (reserved; do not use yet)

### 9.2 Inspect run outcomes

```bash
node scripts/yolo-semantic-gate-report.mjs /path/to/workspace
node scripts/yolo-semantic-gate-report.mjs /path/to/workspace --json
```
