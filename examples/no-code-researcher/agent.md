# Project Agent Guide

Use this file for durable, project-level guidance.

## Default Policy

- Prioritize correctness and traceability over speed.
- Prefer writing concrete deliverables to `outputs/`.
- Keep assumptions explicit and mark unknowns clearly.

## Output Convention

- `outputs/plan.md`: short execution plan and progress.
- `outputs/final-report.md`: final answer/report.
- `outputs/sources.md`: references, links, and evidence trail.

## Quality Bar

- Every major claim should have evidence.
- Avoid overclaiming when source confidence is weak.
- Include a short "limitations" section in final outputs.
