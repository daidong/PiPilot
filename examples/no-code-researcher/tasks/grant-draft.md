# Task Spec: Grant Draft

Goal: Draft a grant proposal skeleton and compliance checklist.

## Required Deliverables

1. `outputs/plan.md`
1. `outputs/grant-outline.md`
1. `outputs/compliance-checklist.md`

## Procedure

1. Identify target agency/program assumptions from user instructions.
1. Use `research-grants` skill scripts when appropriate to scaffold structure.
1. Draft:
   - problem statement
   - objectives / aims
   - method and milestones
   - risk mitigation
   - evaluation plan
1. Build a compliance checklist against explicit requirements.
1. Emit `TASK_COMPLETE` when done.
