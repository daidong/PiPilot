## What is Research Copilot?

An AI-powered desktop research assistant for scientists and academics. Literature search, data analysis, academic writing, cross-project paper memory — powered by your ChatGPT Pro / Claude Max subscription (or an API key).

See the [README "How is Research Copilot different"](https://github.com/daidong/PiPilot/blob/main/README.md#how-is-research-copilot-different-from-claude-cowork) for the full comparison with Claude Cowork.
